Printing House

An application written in Java, showing all of the OOP concepts.

Description

This application demonstrates the work of a real printing house with 2 printing machines that work simultaneously and can print whatever press the user wants - newspapers, magazines, books, posters and etc. The components of this printing house are abstracted in classes and there are showcased most of the OOP principles and concepts.

I've written this app in Java as a university project.
