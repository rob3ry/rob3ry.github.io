Hangman

The original well-known game made in C++

Description

This is a C++ console application, representing the game Hangman. It is text-based with ASCII visualisation of the hanged man. The word to guess is chosen randomly from a file with thousands of words in English. With every guess you either reveal letter(s) of the word or you start building your hangy picture. There's a list of the unused letters to help you know which letters you've already used and which you haven't.
