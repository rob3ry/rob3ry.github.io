Dangerous Dave: Remastered

My first 2D platformer game.

Description

Dangerous Dave: Remastered, as I named it, is a 2D shooter platformer. It is not a fully-functional and finished game, it has many features left to be added or fixed. It consists of only one level, which is enough to showcase all the components needed to construct a 2D platformer - there is gravity, collision detection, integrated camera that follows player movement, animations and state machine for the player and enemies, which also have sort of AI for their behaviour. The level is tile-based, which is the first, but still most used and efficient method of designing and structuring levels in a platformer game. The player in the game must make his way through a wave of enemies, who are trying to kill him, and his objective is to find the final section, which he should interact with to finish the level.

The game is written in Lua using the Love2D game engine.
